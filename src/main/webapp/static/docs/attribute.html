配置产品单元表(UP_PRODUCT_ITEM)
# 字段名 描述 取值
1 PRODUCT_ITEM_ID 产品单元ID ATTR_ID
2 NAME 产品单元名称 NAME
3 ALPHA_NAME 首字母简写 ALPHA
4 ITEM_TYPE 产品单元类型 'ATTRIBUTE_SINGLE'
5 CREATER 创建人 CREATER
6 STATE 产品单元状态 'U'
7 DEL_FLAG 删除标识 '1'
8 CREATE_DATE 创建时间 系统时间
9 EFF_DATE 受理生效时间 系统时间
配置属性表(UP_ATTR)
# 字段名 描述 取值
1 ATTRIBUTE_ID 属性ID ATTR_ID
2 ATTRIBUTE_KIND 属性类型 'ATTRIBUTE_SINGLE'
3 CREATER 创建人 CREATER
4 DEL_FLAG 删除标识 '1'
